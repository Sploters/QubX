## Circuito 2: Estado de Bell (Emaranhamento)

```python
from qiskit import QuantumCircuit
from qiskit_aer import AerSimulator

qc = QuantumCircuit(2, 2)

# Hadamard no qubit 0 вҶ’ superposiГ§ГЈo
qc.h(0)

# CNOT: emaranha os dois qubits
qc.cx(0, 1)

# Mede os dois
qc.measure([0, 1], [0, 1])

sim = AerSimulator()
result = sim.run(qc, shots=1024).result()
counts = result.get_counts()
print(counts)
# Esperado: {'00': ~512, '11': ~512}
# Nunca '01' ou '10' вҖ” os qubits estГЈo emaranhados!
```

O resultado `{'00': ~512, '11': ~512}` demonstra o emaranhamento: os qubits **sempre** sГЈo medidos iguais вҖ” ambos 0 ou ambos 1 вҖ” independente de quantas vezes vocГӘ rodar.

## Circuito 3: Algoritmo de Grover (Busca QuГўntica)

O algoritmo de Grover resolve o problema de **busca em um banco de dados nГЈo estruturado** com vantagem quadrГЎtica:

| Abordagem | Buscas necessГЎrias |
|-----------|-------------------|
| ClГЎssico (pior caso) | N buscas |
| QuГўntico (Grover) | вҲҡN buscas |

Para 1 milhГЈo de itens: clГЎssico = 1.000.000 buscas, Grover = ~1.000.

```python

from qiskit_aer import AerSimulator

# Passo 1: Hadamard em todos вҖ” superposiГ§ГЈo uniforme

# Passo 2: OrГЎculo вҖ” marca |11вҹ© com fase negativa

# Passo 3: DifusГЈo вҖ” amplifica o estado marcado

# MediГ§ГЈo

# Esperado: {'11': ~1024} вҖ” Grover amplificou o estado correto!
```
